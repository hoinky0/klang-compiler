# klang-compiler

항목,핵심 내용 및 작성 가이드
1.1 언어 이름 및 정의,언어 이름: K-Lang (한글 프로그래밍 언어)  유형: 교육용/명령형 언어의 AST Interpreter
1.2 설계 동기 및 목표,"동기: 컴파일러 이론 학습과 한글 키워드 사용을 통한 독창성 확보.  목표: 변수, 산술, 조건문, 반복문 등 필수 제어 흐름을 오류 없이 처리하는 Interpreter 구현."
1.3 주요 특징,"* 한글 키워드: 사용자 친화적인 한글 예약어 사용 (만약, 출력, 정수).  * 단일 자료형: 정수(정수) 자료형만을 지원하여 복잡도 최소화.  * AST 기반 실행: 소스 코드를 AST로 변환 후 직접 해석하여 실행."
